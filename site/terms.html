2. Description of Services

AgentRails is an AI agent payment infrastructure platform. We provide the tools, APIs, and governance layer that enable AI agents to autonomously transact using the x402 payment protocol. Our Services include:

  • SaaS Platform: A cloud-hosted platform available across multiple subscription tiers (Sandbox, Pay-as-you-go, Pro, and Enterprise) for building, deploying, and managing AI agent payment infrastructure.
  • x402 Payment Protocol: An implementation of the HTTP 402 payment standard that enables AI agents to make autonomous USDC stablecoin payments on supported blockchain networks (Base, Ethereum, and Arc).
  • Agent Management Dashboard: A web-based interface for managing AI agents, wallets, spending policies, and transaction histories.
  • Open Source SDKs: Client libraries for integrating x402 payments into AI agent frameworks, including LangChain, CrewAI, Semantic Kernel, and Microsoft Agent Framework.
  • Consulting Services: Enterprise architecture, Copilot Studio buildout, MCP server development, and agent governance consulting.
  • API and Sandbox: REST APIs and a sandbox environment for development, testing, and integration.

5. Blockchain Transactions

The x402 payment protocol facilitates USDC stablecoin transactions on public blockchain networks, including Base, Ethereum, and Arc. By using these features, you acknowledge and agree to the following:

5.1 Irreversibility

Blockchain transactions are irreversible once confirmed and settled on-chain. AgentRails cannot reverse, cancel, or modify a transaction after it has been submitted to and confirmed by the blockchain network. You are solely responsible for verifying transaction details (recipient address, amount, and network) before authorizing a payment.

5.2 Public Nature

Blockchain transactions are recorded on public, immutable ledgers. Transaction details -- including wallet addresses, amounts, and timestamps -- are publicly visible. AgentRails does not control the public visibility of on-chain data.

5.3 Gas Fees

Executing transactions on blockchain networks requires gas fees paid to network validators. These fees are determined by network conditions and are not controlled by AgentRails. Gas fees are non-refundable regardless of whether the underlying transaction succeeds or fails.

5.4 Network Risks

Blockchain networks may experience congestion, outages, forks, or protocol changes that affect transaction processing. AgentRails is not responsible for delays, failures, or losses arising from blockchain network conditions.
